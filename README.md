## Converting HTML to PDF

### Method 1: Using Your Browser (Recommended - Easiest!)
1. Open `bourbon-presentation.html` in any web browser (Chrome, Safari, Firefox, Edge)
2. Go to File → Print (or press Cmd+P on Mac, Ctrl+P on Windows)
3. In the print dialog:
   - Select "Save as PDF" as the destination/printer
   - **IMPORTANT: Set Paper Size to "Tabloid" or "11x17" or custom size 16x9 inches** (for landscape widescreen format)
   - Set Margins to "None" or "Minimum"
   - Make sure "Background graphics" is enabled
4. Click "Save" and name it `bourbon-tasting-presentation.pdf`

**Chrome Users (Best Results):**
- Open Developer Tools (Cmd+Option+I)
- Click the 3 dots menu → More tools → Rendering
- Under "Emulate CSS media type" select "Print"
- Then use File → Print → Save as PDF

### Method 2: Using Safari (Mac - Very Good Quality)
1. Open `bourbon-presentation.html` in Safari
2. File → Export as PDF
3. Save as `bourbon-tasting-presentation.pdf`

### Method 3: Online Converter
- Upload the HTML file to: https://www.html2pdf.com/ or https://cloudconvert.com/html-to-pdf

## Presentation Tips

### Viewing the Presentation
- The HTML file works great fullscreen in any browser (press F11 or Fn+F)
- Use arrow keys or Page Down/Up to navigate slides
- The PDF is easier to advance through during live presentation

### Before the Event
1. Review the presenter notes thoroughly
2. Print the quick reference sheet for easy access
3. Test the presentation on your display equipment
4. Have water available for bourbon tasting
5. Arrange food pairings on the table

### During the Event
- Keep pacing relaxed (approximately 2-3 minutes per bourbon section)
- Encourage questions and discussion
- Let guests smell before tasting
- Remind guests to cleanse palate between pours
